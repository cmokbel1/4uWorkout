This (Insert App Name) is a dual working excercise and music system. Users can sort through varying workouts in relation to specific body parts, and songs in relation to specific genres. Create a random workout list with our randomized workout button and then saving it OR create a random playlist with out randomized music button! Users also have the option to use a search bar with accepted inputs to find workouts for specific body parts. The music playlist can be created randomly, or the user has the option to select a genre and generate a playlist of the top 50 songs in that genre.

Our project makes use of the excerciseDB as our first API from the rapid API website. The second API that we use is the shazam core API from the aforementioned website. The excerciseDB is comprised of roughly 1500 different workouts so the user can be certain that their workout will be dynamic for the foreseeable future. Simiarly, the shazam core is regularly updated array of the top 50 songs in each of the available genres.
